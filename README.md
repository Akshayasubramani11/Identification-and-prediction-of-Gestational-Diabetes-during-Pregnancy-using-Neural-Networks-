# Identification-and-prediction-of-Gestational-Diabetes-during-Pregnancy-using-Neural-Networks-
Developed an AI-based Gestational Diabetes Prediction System using Neural Networks to identify diabetes risk during pregnancy at early stages.
Implemented FNN, CNN, RNN, and Hybrid CNN-RNN models using Python and Machine Learning techniques for predictive analysis.
Performed data preprocessing, feature selection, scaling, and model evaluation on clinical and lifestyle healthcare datasets.
Achieved high prediction accuracy of 97% using the Hybrid Neural Network model for effective risk assessment.
Integrated the trained model into a Flask web application for real-time diabetes prediction and user-friendly healthcare support.
Focused on improving maternal and fetal health outcomes through intelligent early diagnosis and personalized healthcare solutions.
